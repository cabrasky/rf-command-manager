📥 Pull Commands from Home Assistant

Broadlink learned commands are stored in the HA container filesystem. The REST/WebSocket API does not expose them — you need to extract the file manually.

/config/.storage/broadlink_remote_e87072ba795c_codes

Option 1 — SSH into node1:
docker exec homeassistant cat /config/.storage/broadlink_remote_e87072ba795c_codes

Option 2 — File Editor addon: Open the File Editor addon in HA, navigate to the path above, copy the contents.

Option 3 — VS Code Server addon: Browse to the .storage folder and open the file.
